# 닉네임·태그·프로필 노출 플레이북

지식iN에서 직접 브랜드명을 언급하지 않고 노출·신뢰를 쌓는 3가지 간접 채널과 그 조합 전략.

## 1. 닉네임 전략

### 1-1. 좋은 닉네임의 조건

- **직무·업계 암시**: "B2B 세일즈 컨설턴트", "그로스 마케터 12년차"
- **일관성**: 한 번 정하면 계정 수명 내내 유지
- **겸손·신뢰**: "최고", "지존" 같은 자뻑보다 "초보 도우미", "현직자"가 채택률이 높다 (실사용 데이터)
- **한글 우선**: 영문 닉네임은 노출·가독성 모두 떨어짐

### 1-2. 피해야 할 닉네임

- ❌ 특정 회사명 포함 (`OO컴퍼니대표`)
- ❌ 자격증 허위 표기 (`변호사OO` — 실제 자격 있을 때만)
- ❌ 너무 일반적 (`마케팅`, `전문가` — 노출 경쟁 치열)
- ❌ 숫자·특수문자 남발

### 1-3. 닉네임 예시 (업계별)

| 업계 | 추천 패턴 |
|------|-----------|
| SaaS B2B | "B2B 세일즈 12년차", "SaaS 성장 실무자" |
| 이커머스 | "쇼핑몰 운영 8년차", "온라인 셀러 도우미" |
| 마케팅 에이전시 | "디지털 광고 현직자", "퍼포먼스 마케팅 실무" |
| 교육·코칭 | "영어 학습 10년 연구자", "취업 컨설턴트" |
| 법률·세무 (자격증 必) | "변호사 OO", "세무사 OO" |
| 부동산 | "공인중개사 15년", "임대차 실무 현직자" |

## 2. 태그 전략

### 2-1. 태그의 가중치

지식iN 검색 결과 노출에서 태그는 본문보다 약한 신호지만, **질문-답변 클러스터링**에 영향을 준다. 같은 태그가 달린 Q&A끼리 "관련 질문"으로 상호 노출되므로 업계 태그를 일관되게 쌓으면 노출 면적이 넓어진다.

### 2-2. 태그 배치 규칙

- **답변당 3-5개**: 너무 적으면 노출 기회 손실, 너무 많으면 스팸처럼 보임
- **일반 > 브랜드 비율 4:1**: 일반 업계 키워드가 대부분, 브랜드 키워드는 1개 이하
- **한글 표준 용어**: 영문·약어는 한글 버전과 병기 (`#SaaS #서비스형소프트웨어`)
- **트렌드 반영**: 월 1회 검색 추세 확인해 태그 교체

### 2-3. 태그 예시 (SaaS B2B 업계)

**안전 조합 (브랜드 없음)**:
`#SaaS #B2B마케팅 #온라인마케팅 #스타트업`

**브랜드 포함 조합 (1개만)**:
`#SaaS #B2B마케팅 #마케팅자동화 #OO브랜드 #퍼포먼스마케팅`
→ 5개 중 1개가 브랜드. 브랜드 태그가 답변 광고성을 강화하지 않도록 다른 일반 태그들 사이에 배치.

### 2-4. 피해야 할 태그 패턴

- ❌ 브랜드 태그 2개 이상 (스팸 신고 위험↑)
- ❌ 자사·경쟁사 태그 동시 (의도 불분명·부정적 인상)
- ❌ 특가·이벤트성 태그 (`#할인`, `#쿠폰`, `#무료`) — 영리 목적으로 즉시 분류

## 3. 프로필 전략

### 3-1. 프로필 구성 요소

- **닉네임**: 2절 참조
- **프로필 사진**: 실제 얼굴 > 일러스트 > 제품 이미지 (신뢰 순)
- **한 줄 소개**: 직무·전문 분야 간결하게 (예: "B2B SaaS 성장 전략 12년. 주로 퍼널·리텐션·프라이싱 답변합니다.")
- **상세 소개**: 경력·전문 분야·답변 톤
- **관심 카테고리**: 3-5개로 집중 (답변이 특정 분야에 몰려야 등급·신뢰 상승)

### 3-2. 프로필에서 할 수 있는 것 vs 할 수 없는 것

| 가능 | 불가 |
|------|------|
| 업계·직무 명시 | 특정 회사명 명시 (회색지대) |
| 전문 분야 요약 | 자사 웹사이트 직접 링크 |
| 경력 연수 | 영업 이메일·연락처 노출 |
| 답변 스타일 설명 | 유료 상담 안내 (엑스퍼트 등록 없이는) |

### 3-3. 기업 엑스퍼트 경로

기업이 투명하게 운영하려면 [네이버 엑스퍼트](https://expert.naver.com) 등록이 정공법. `naver-kin-policy.md`의 Section 4 참조.

## 4. 등급 성장 전략

### 4-1. 등급별 성장 단계 (일반적 패턴)

지식iN의 구체적 등급명·내공 구간은 네이버 정책에 따라 변경되므로 [지식iN 등급 공식 안내](https://help.naver.com/alias/kin/kin_1_4.naver)에서 현재 기준을 확인한다. 일반적 성장 단계는 다음과 같다.

| 단계 | 일반적 특징 |
|------|-------------|
| 초기 | 답변 노출·신뢰 낮음. 등급 표기로 인한 필터링에 걸리기 쉬움 |
| 중급 | 답변 노출 시작. "중수 이상" 필터를 사용하는 질문자에게 노출 |
| 상급 | 노출·신뢰 급상승. 검색 결과 상단 편성 확률 증가 |
| 최상위 | "영향력 있는 답변자"로 가시화. 채택률·프로필 클릭 상승 |

내공 구간과 등급명은 수시 변경되므로 위 표는 참고용. 기업은 일정 단계 이상 도달 후 엑스퍼트 등록으로 전환하는 것이 효율적.

### 4-2. 내공 획득 경로

- **채택**: 1 채택 = 25 내공
- **추천**: 1 추천 = 2 내공
- **답변 자체**: 1 답변 = 2 내공
- **질문 작성**: 1 질문 = 1 내공 (자문자답 방지 목적으로 낮음)

### 4-3. 빠른 성장 전략

1. **쉽고 구체적인 질문에 빠르게 답변**: 답변 0-1개인 질문, 구체적 상황 포함 질문 → 채택률 高
2. **같은 카테고리에 집중**: 분산하면 신뢰도 분산
3. **답변 품질 > 양**: 일 3-5개 고품질 답변 > 일 20개 저품질
4. **이미지·표 활용**: 지식iN 답변에 이미지 업로드 가능, 가시성·채택률 모두 상승

## 5. 조합 시나리오

### 시나리오 A: B2B SaaS 스타트업 (6개월 목표)

- **닉네임**: "B2B SaaS 성장 실무자"
- **주 카테고리**: 컴퓨터통신 > 소프트웨어, 비즈니스/창업 > 마케팅
- **태그 조합**: `#SaaS #B2B마케팅 #그로스 #스타트업 #퍼포먼스마케팅`
- **답변 주제**: 마케팅 자동화 툴 선택, 지표 설정, 퍼널 개선 (일반론만)
- **목표**: 6개월 내 영웅 등급 → 자사 FAQ·블로그로 Q&A 자산화 200건

### 시나리오 B: 법무법인 (엑스퍼트 경로)

- **엑스퍼트 등록**: 변호사 자격증 인증 → 공식 엑스퍼트
- **프로필**: 법무법인명 공식 표기 (엑스퍼트는 투명성 보장 경로)
- **유료 상담**: 일부 질문에 유료 상담 연결 (광고 표기 의무)
- **일반 답변**: 무료 답변으로 브랜드 신뢰 축적

### 시나리오 C: 이커머스 셀러 (커뮤니티 확장)

- **닉네임**: "쇼핑몰 운영 8년차"
- **태그**: `#온라인쇼핑몰 #스마트스토어 #셀러 #이커머스`
- **답변 주제**: 배송·CS·스마트스토어 운영 일반론
- **확장**: 지식iN → 네이버 카페·블로그 일관 브랜딩 (`community-marketing` 스킬 연계)

## 6. 체크리스트

닉네임·프로필·태그 세팅 전 확인:

- [ ] 닉네임이 브랜드명·회사명을 암시하지 않는가?
- [ ] 프로필에 외부 링크가 없는가?
- [ ] 관심 카테고리가 3-5개로 집중되어 있는가?
- [ ] 태그 전략이 일반:브랜드 = 4:1 비율을 지키는가?
- [ ] 엑스퍼트 등록을 검토했는가? (기업 운영 시 강력 권장)
