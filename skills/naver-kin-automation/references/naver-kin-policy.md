# 네이버 지식iN 운영정책 요약 (마케팅 관점)

> 이 문서는 2026년 기준 공개 운영정책을 마케팅 업무 관점에서 발췌·요약한 것이다. 원문은 [네이버 지식iN 운영원칙](https://help.naver.com/alias/kin/kin_11.naver)과 [네이버 이용약관](https://policy.naver.com/policy/service.html)을 확인해야 한다. 정책은 언제든 개정될 수 있으므로 캠페인 시작 전 재확인 필수.

## 1. 금지 행위 (High-Risk)

마케팅 캠페인에서 가장 빈번한 위반 유형.

### 1-1. 광고·홍보·영리 목적 답변
- 자사 제품·서비스를 추천·알선하는 답변
- 답변에 자사 웹사이트·쇼핑몰·SNS 링크 삽입
- 쿠폰·할인 코드·이벤트 안내
- **제재**: 답변 삭제 → 활동 정지 (누적 시 영구)

### 1-2. 도배·반복 답변
- 여러 질문에 같은 템플릿 답변 복붙
- 단기간 대량 답변 등록 (시간당 수십 건 이상)
- **제재**: 스팸으로 분류, 계정 제한

### 1-3. 자문자답·다계정 조작
- 본인 또는 팀이 질문을 올리고 다른 계정으로 답변
- 채택률을 인위적으로 올리려는 행위
- **제재**: 영구 정지, IP 차단 가능

### 1-4. 타인 사칭·허위 경력
- 존재하지 않는 자격증·경력 표기
- 의사·변호사 등 전문직 사칭 (법적 문제까지 확대)
- **제재**: 영구 정지, 사법 처벌 가능

### 1-5. 비방·명예훼손
- 근거 없는 경쟁사 깎아내리기
- 특정 브랜드·개인에 대한 허위 사실 유포
- **제재**: 삭제·정지 + 민형사상 책임

## 2. 주의 영역 (Gray Zone)

법적으로는 허용되지만 신고·정지 위험이 높다.

### 2-1. 외부 링크
- 자사 링크 직접 삽입은 금지
- 위키피디아·나무위키·정부기관 링크는 보조 자료로 사용 가능
- **원칙**: 링크는 본문 정보의 보조 근거일 때만, 답변당 최대 1개

### 2-2. 간접 광고
- "저는 OO 분야 전문가인데요" 식의 자기 소개 후 일반 조언
- 닉네임·프로필에 업계·직무 표기
- **경계선**: 본문에서 브랜드명을 지칭하지 않으면 대체로 허용. 하지만 동일 계정이 계속 같은 업계 질문에만 답하면 "영업 목적 계정"으로 신고될 수 있다.

### 2-3. 카테고리 무관 답변
- 본인 전문 카테고리가 아닌 곳에 답변
- 노출은 늘지만 신뢰도 하락

### 2-4. 답변 수정·삭제
- 채택 후 답변 내용 변경은 질문자가 위화감을 느끼고 신고로 이어질 수 있음
- 수정은 맞춤법·명확성 개선 수준에서만

## 3. 권장 행위 (Green)

지식iN 내 자산을 건강하게 쌓는 길.

### 3-1. 등급 시스템 활용
- **등급 진행**: 평민 → 시민 → 중수 → 고수 → 영웅 → 수호신 → 초인 → 태양신 (대략)
- 등급 상승 조건: 답변 수, 채택률, 내공
- 높은 등급 = 노출·신뢰 모두 상승

### 3-2. 엑스퍼트 제도
- 의사·변호사·세무사 등 전문 자격증 보유자 인증 시스템
- 기업 계정도 "기업 엑스퍼트"로 등록 가능 (단, **공식 채널 등록**이 필요하며, 이 경우는 광고 표기가 투명하게 처리됨 → 정책상 권장 경로)
- 일반 기업 마케팅 팀에게는 엑스퍼트 등록이 가장 정공법

### 3-3. Q&A → 자산화
- 답변은 지식iN 소유(약관상 네이버가 이용권을 가짐)이지만, **질문+자신의 답변의 재구성**은 가능
- 자사 블로그·FAQ로 재편성해 브랜드 자산으로 축적

## 4. 엑스퍼트 등록 절차 (권장 경로)

기업이 정공법으로 지식iN에서 활동하려면 **엑스퍼트 등록**이 가장 안전하다.

1. [네이버 엑스퍼트](https://expert.naver.com) 가입
2. 기업 정보·사업자등록증 제출
3. 분야 선정 (마케팅 업계라면 "마케팅/광고" 카테고리)
4. 승인 후 프로필에 공식 "기업 엑스퍼트" 배지 부여
5. 유료 상담·답변 제공 가능 (광고 표기 의무, 투명성 확보)

**장점**: 광고 표기를 투명하게 하면서 브랜드 노출 가능. 일반 계정의 회색지대 운영보다 ROI·지속성 모두 우수.

## 5. 신고 프로세스

지식iN에서 다른 사용자가 답변을 신고할 수 있다.

1. 신고 접수 → 네이버 검수팀 수동·자동 검토 (24-72시간)
2. 위반 확정 시 답변 삭제 + 활동 정지 (누적 횟수별 차등)
3. 이의제기는 1회 가능하나 인용률 낮음

**누적 기준** (대략):
- 경고 1-2회: 답변 삭제만
- 3-5회: 단기 정지 (7-30일)
- 6회 이상: 장기·영구 정지

## 6. 체크리스트: 답변 등록 직전

- [ ] 제품·브랜드명이 본문에 나오지 않는가?
- [ ] 외부 링크가 0개이거나 공공·중립 출처 1개 이하인가?
- [ ] 이 질문에 이미 자사 계정이 답변하지 않았는가?
- [ ] 최근 1시간 이내 같은 템플릿 답변을 올리지 않았는가?
- [ ] 카테고리가 프로필 전문 분야와 맞는가?
- [ ] 면책 문구가 필요한 주제(법률·의료·금융)인가? 필요 시 삽입했는가?
- [ ] 경쟁사 언급이 있다면 사실에 기반하고 근거를 제시하는가?

## 7. 자주 묻는 질문 (내부)

**Q. 닉네임에 "OO회사 대표" 넣어도 되나?**
A. 대표·직원임이 드러나는 닉네임으로 업계 답변을 계속 달면 자사 영업 계정으로 분류될 위험이 크다. "업계 12년차" 같은 간접 표기가 안전하다.

**Q. 답변에 "제가 일하는 회사에서는..."은 괜찮나?**
A. 회색지대. 구체적 회사명을 생략하면 대체로 허용되지만, 답변 맥락에서 브랜드가 유추 가능하면 신고 위험. 일반화된 "업계 관행" 형태로 풀어 쓰는 것을 권장.

**Q. AI로 답변을 생성하면 정책 위반인가?**
A. 지식iN은 명시적으로 "AI 생성 답변 금지"를 규정하지 않지만, **실질적으로 생성형 답변은 내공 확보 행위로 의심받을 수 있다.** 더 중요한 건 자동 등록(봇)이 금지라는 점이다. AI를 초안으로 쓰고 사람이 검토·수정·수동 등록하는 방식은 정책상 문제되지 않지만, 본 스킬은 자동 등록 자체를 지원하지 않는다.
