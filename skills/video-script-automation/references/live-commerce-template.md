# 라이브커머스 대본 템플릿

네이버쇼핑라이브·쿠팡라이브·카카오쇼핑라이브의 60분 표준 방송 구조와 한국 주요 플랫폼 차이.

## 플랫폼별 특징

| 항목 | 네이버쇼핑라이브 | 쿠팡라이브 | 카카오쇼핑라이브 |
|------|-------------------|------------|-------------------|
| 주 시청자 | 3040 여성·가족 | 2030 직장인 | 2030·카카오톡 이용자 |
| 커머스 강점 | 스마트스토어 연동·알림 | 쿠팡 회원·로켓배송 | 카카오톡 공유·친구 초대 |
| 쿠폰·혜택 특성 | 네이버페이 적립·라이브 전용 쿠폰 | 쿠폰+로켓배송 심리 | 카카오톡 친구 초대 리워드 |
| 권장 방송 길이 | 60-90분 | 60분 | 45-60분 |
| 채팅 반응 유도 | 「N 댓글 터지면 쿠폰」 패턴 | 좋아요 마일스톤 | 카톡 공유 미션 |
| 호스트 스타일 | 친근·정보 중심 | 빠른 템포·가격 강조 | 라이프스타일·감성 |

## 60분 표준 타임라인

### 0-5분 : 오프닝 (Cold Start Drop 방지)

**목표**: 시청자 이탈 차단, 체류 동기 즉시 제공

```
호스트: "안녕하세요! OO브랜드 라이브에 오신 여러분 환영합니다.
        저는 오늘 방송을 진행할 [이름]입니다.

        지금 들어오신 분들, 꼭 끝까지 함께 해주세요.
        왜냐하면 마지막 10분에 오늘 방송에서만 공개되는
        30% 추가 쿠폰 코드가 나옵니다.

        먼저 오늘 준비한 상품 간단하게 보여드릴게요."
```

**필수 요소**:
- 방송 총 길이 암시 ("1시간 동안")
- 체류 보상 예고 (쿠폰·선물·사은품)
- 3상품 or 3옵션 프리뷰 (스크롤 멈춤 유도)
- 「광고」 또는 「유료광고 포함」 자막 고정 (전자상거래법)

### 5-20분 : 상품 1 심층 소개

**구조**: 보여주기 → 설명 → 사용 시연 → 후기 → 가격

```
## 보여주기 (5-7분)
- 카메라 클로즈업, 360도 회전, 크기 비교물(동전·A4)
- 색상·소재 감각적 묘사 ("물에 손 담근 듯한 질감")

## 설명 (7-12분)
- 문제 제기 → 해결 포인트 3개
- 기능 시연 (실제 사용 모습)
- 타겟 상황 ("퇴근 후 15분이 바뀝니다")

## 사용 시연 (12-16분)
- 호스트 직접 사용, 효과 시각화
- before/after (식품·뷰티는 필수)

## 고객 후기 (16-18분)
- 스크린샷·인용 3-5개 (동의받은 것만)
- 리뷰 키워드 강조 자막

## 가격·혜택 (18-20분)
- 정가 → 방송가 → 쿠폰 적용가 단계별 공개
- "지금 구매 이유" 3가지 (재고 한정·당일 배송·방송 특별가)
```

**Q&A 프롬프트**: "댓글로 궁금한 점 남겨주세요. 5개 이상 달리면 바로 답변드립니다."

### 20-25분 : 실시간 Q&A 1차

**필수 답변 질문 (예상)**:
- "사이즈 어떻게 나와요?" → 착용 비교
- "피부 타입별로 괜찮나요?" → 전성분 공개
- "해외 배송 되나요?" → 배송 조건 명시
- "할인 쿠폰 어디서 받나요?" → 현재 적용된 쿠폰 화면 고정

**Tip**: 부정적 질문 (불량·환불)도 회피하지 않고 정책 그대로 답변. 회피하면 신뢰 급락.

### 25-40분 : 상품 2·3 또는 옵션 다양화

같은 구조 반복하되 단조로움 방지를 위해 변주.

- 상품 2는 **비교 구도** ("상품1이 OO 타입이면 이건 ㅁㅁ 타입")
- 상품 3는 **번들 제안** ("2개를 묶으면 가격이 이렇게 됩니다")

### 40-45분 : 실시간 Q&A 2차 + 미구매자 타겟 멘트

아직 구매 버튼 누르지 않은 시청자 대상 설득.

```
호스트: "아직 고민 중이신 분들, 이 상품 지금 구매하시면
        내일 도착합니다. 5일 뒤 모임에 딱 맞춰 받을 수 있어요.
        지금 방송 쿠폰 합치면 정가 대비 42% 할인인데,
        내일 자정에 쿠폰이 만료됩니다."
```

### 45-55분 : 사용 시연 심화 + 특이 사용 사례

- 뷰티: 다른 타입·연령대 모델로 연속 시연
- 가전: 실제 가정 상황 (소음·전기·공간) 측정치 공개
- 패션: 여러 코디 조합 빠른 전환
- 식품: 실제 시식·요리 활용법

### 55-60분 : 한정 특가·쿠폰 릴리즈·클로징

**최대 클라이맥스 구간**.

```
호스트: "드디어 약속한 특가 공개합니다!

        지금부터 10분 동안만, 아래 쿠폰 코드 입력하시면
        방송가에서 추가 30% 할인 들어갑니다.

        코드는 LIVE30 입니다. 채팅창에도 띄워드릴게요.

        재고 [남은 수량]개, 선착순이에요.
        방송 끝나면 이 혜택 사라집니다."

[화면 하단 자막: LIVE30 / 23:45까지 / 재고 47개]
```

**클로징**:
- 감사 인사
- 다음 방송 예고·알림 설정 유도 (네이버쇼핑라이브 알림받기·쿠팡 찜하기)
- 브랜드 SNS 유도 (카카오톡 채널·인스타)

## 호스트 필수 멘트 팩

### 체류 유도 (10분마다)
- "지금 들어오신 분들 환영합니다. 방송 끝 10분 전에 특가 쿠폰 있어요."
- "채팅 올려주세요, 50개 넘으면 사은품 추첨합니다."

### 구매 촉구 (상품 설명 후)
- "재고 [N]개 남았어요. 방송 끝나면 이 가격 사라집니다."
- "지금 주문하시면 내일 로켓배송으로 받을 수 있어요."

### 불안 해소
- "반품은 100% 가능합니다. 7일 이내 무료 반품."
- "사이즈 안 맞으면 무료로 교환해드려요."

### 실시간 반응
- "OO님, 질문 감사해요. 답변드릴게요."
- "방금 30분 동안 [N]개 팔렸어요. 재고 [M]개 남았습니다."

## 컴플라이언스 체크 (방송 전·후)

### 방송 전
- [ ] 「광고」 자막 고정 배치
- [ ] 사업자 정보 상시 노출 (전자상거래법 필수)
- [ ] 통신판매업 신고번호 표시
- [ ] 쿠폰·할인가 표기가 정가 명확히 보여주는가 (「정가 X원 → 방송가 Y원」)
- [ ] 식품·건강식품·화장품: 식약처·공정위 문구 점검

### 방송 중
- [ ] 청약철회·환불 정책 1회 이상 구두 고지
- [ ] 효능·효과 과장 표현 사용 안 함 ("최고", "유일", 검증 없는 수치)
- [ ] 타 브랜드 특정·비방 없음
- [ ] 시청자 실명·개인정보 화면 노출 안 함

### 방송 후
- [ ] VOD 업로드 시 동일 컴플라이언스 유지
- [ ] 방송 성과 리포트 (시청자·전환·매출) 내부 공유
- [ ] 후기 수집 (다음 방송 활용)
