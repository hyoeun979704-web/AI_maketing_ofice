# 숏폼 대본 플레이북

YouTube Shorts·Instagram Reels·TikTok용 15-60초 대본 구조와 한국어 자막 최적화.

## 플랫폼별 특성

| 항목 | YouTube Shorts | Instagram Reels | TikTok |
|------|-----------------|------------------|--------|
| 길이 | 최대 60초 (15-30초 최적) | 최대 90초 (7-15초 최적) | 최대 10분 (9-30초 최적) |
| 알고리즘 | 체류·반복 시청 중시 | 초기 3초 이탈 중요 | 공유·완주율 중시 |
| 주 시청자 | 전 연령 | 2030 여성 多 | 10대·20대 |
| 트렌드 오디오 | 영향 낮음 | 영향 중 | 영향 높음 (필수에 가까움) |
| 링크 | 설명·고정 댓글 | 스티커·프로필 | 프로필 링크 |

## 숏폼 3초 룰

첫 3초에 **훅 + 스크롤 정지 이유**가 동시에 나와야 한다.

### 훅 6대 카테고리

| 카테고리 | 원리 | 예시 |
|----------|------|------|
| 질문 | 호기심 자극 | "월 100만원으로 OO 가능할까요?" |
| 수치 충격 | 인지 부조화 | "3개월 만에 매출 2배, 방법은 단 3가지" |
| 역발상 | 통념 반전 | "세일즈는 말을 덜 할수록 잘 됩니다" |
| 호기심 격차 | 정보 공백 | "이거 모르면 1000만원 손해봅니다" |
| 즉시 유용 | 실용성 | "지금 당장 쓸 수 있는 3가지" |
| 감정 공감 | 동일시 | "이번 분기 목표 못 채웠죠?" |

### 훅 제작 체크

- [ ] 3초 안에 훅 완결되는가?
- [ ] 자막·영상·음성 중 최소 둘이 훅을 동시에 전달하는가?
- [ ] 타겟이 "어? 뭐지?" 할 정도의 인지 부조화가 있는가?
- [ ] 훅이 본문 내용과 실제로 연결되는가? (낚시 금지 — 완주율 하락)

## 구조 템플릿 (30초 기준)

```
0-3초:   Hook
3-8초:   Context (왜 이게 중요한가)
8-22초:  Body (핵심 내용)
22-27초: Takeaway (요약)
27-30초: CTA
```

### 짧은 버전 (9-15초)

```
0-3초:  Hook
3-10초: 핵심 1개 문장 + 시연
10-15초: CTA
```

### 긴 버전 (45-60초)

```
0-3초:   Hook
3-8초:   Context
8-20초:  포인트 1
20-35초: 포인트 2
35-50초: 포인트 3
50-55초: Summary
55-60초: CTA
```

## 한국어 자막 규격

### 음절 속도
- 1초당 4-6음절 (편안한 읽기)
- 1초당 7음절 이상은 속독 스트레스
- 내레이션 속도와 일치시킬 것

### 한 줄 길이
- 최대 14자 (모바일 세로 화면 기준)
- 2줄 이상은 화면 하단 1/3 가리므로 터치 방해
- 긴 문장은 2장면으로 분할

### 폰트·크기
- 한글 고딕 계열 Bold (노토 산스 / 본고딕)
- 크기: 화면 높이의 5-7%
- 외곽선 + 그림자로 배경 영상 변화에 대응
- 색: 흰색 기본, 강조는 노랑·밝은 민트

### 위치
- 하단 1/4 지점 (UI 버튼과 겹치지 않음)
- 강조 키워드는 화면 중앙 일시 표시 (1초 이내)

### 자막 스타일 예시

```
[일반 자막]
하단 1/4, 흰색 + 검정 외곽선, 14자 1줄

[강조 자막]
화면 중앙, 노랑 + 검정 외곽선, 굵게
0.5-1초 짧게 플래시

[데이터 자막]
화면 상단, 숫자 강조 색상
애니메이션 효과 (카운트업)
```

## 예시: 3가지 포맷

### 예시 A: 교육·정보 (YouTube Shorts 30초)

```
[0-3초] Hook
화면: 화자 정면, 깜짝 놀란 표정
자막: "당신의 영상이 조회수가 안 나오는 진짜 이유"
내레이션: "조회수 안 나오는 이유, 썸네일이 아닙니다."

[3-8초] Context
화면: 썸네일 클릭율 차트 예시
자막: "95%가 여기서 실수합니다"
내레이션: "진짜 이유는 첫 3초에 있어요."

[8-22초] Body
화면: Before/After 썸네일·훅 비교
자막: [각 포인트별 짧게]
  "1. 훅이 구체적"
  "2. 화면 전환이 3초 이내"
  "3. 음성보다 자막이 먼저"
내레이션: "첫째, 훅이 구체적이어야 합니다..."

[22-27초] Takeaway
자막: "3초에 승부"
내레이션: "이 3가지만 바꿔도 조회수 2배."

[27-30초] CTA
화면: 화자 + 프로필 클로즈업
자막: "프로필에서 템플릿 받아가세요"
내레이션: "템플릿 프로필에 있어요."
```

### 예시 B: 제품 시연 (Instagram Reels 15초)

```
[0-3초] Hook + Problem
화면: 지저분한 옷장 대참사
자막: "매일 아침 이 지옥"
(트렌드 오디오: 지지직 사운드)

[3-10초] Solution
화면: 슬림 옷걸이 적용 (빠른 타임랩스)
자막: "이 옷걸이 하나로"
내레이션: "공간 2배, 꺼내기 3배 빠르게."

[10-15초] CTA
화면: 제품 + "-50%"
자막: "프로필 링크"
내레이션: "오늘까지만 반값."

고정: 「광고」 (우상단)
```

### 예시 C: 감성·스토리 (TikTok 9초)

```
[0-3초] Hook
화면: 새벽 5시, 노트북 앞 창업자
자막: "3번째 실패 후"
트렌드 오디오: [감성 K팝]

[3-6초] Turn
화면: 제품 대시보드에서 첫 매출 알림
자막: "드디어 돌아옵니다"

[6-9초] CTA
자막: "당신의 시작, 우리가 돕습니다"
화면: 브랜드 로고 + 링크

고정: 「광고」
```

## 트렌드 오디오 활용 (TikTok·Reels)

### 활용 원칙
- **TikTok**: 트렌드 오디오 사용 시 노출량 2-3배. 트렌드 페이지에서 "상승 중" 필터
- **Instagram Reels**: 트렌드 오디오 영향 중간. 자체 오디오로도 가능
- **YouTube Shorts**: 트렌드 영향 낮음. 오리지널 오디오 선호

### 선택 기준
- 우리 브랜드 톤에 맞는가
- 가사·메시지가 광고 내용과 충돌하지 않는가
- 저작권 라이선스 이슈 없는가 (플랫폼 공식 라이브러리만 사용)

### 한국 트렌드 오디오 패턴 (2026 기준)
- 뉴진스·아이브·르세라핌 등 K팝 후크 반복
- 한국 드라마·예능 명대사 밈
- ASMR·사운드 이펙트 (지지직·Ding 등)
- 레트로 BGM (7080·90년대 광고 사운드 리바이벌)

## CTA 패턴 (숏폼)

| 목적 | 권장 CTA |
|------|----------|
| 프로필 방문·팔로우 | "팔로우하고 다음 영상 놓치지 마세요" |
| 링크 클릭 | "프로필 링크에서 확인" |
| 댓글 유도 | "댓글에 '질문' 남겨주세요, 답변드립니다" |
| 공유 | "이거 필요한 친구 태그" |
| 저장 | "나중에 쓸 수 있게 저장하세요" |

**TIP**: 여러 CTA를 섞지 말 것. 한 영상 = 한 CTA.

## 완주율·재시청 유도 테크닉

### 완주 유도 (Retention)
- 첫 3초에 "이 영상은 X로 끝납니다" 예고 (호기심 지속)
- 중간에 텍스트 빨리 지나가게 해 일시정지·재시청 유도
- 마지막에 "처음 봤을 때 놓친 거 있죠?" 멘트로 재시청 유도

### 재시청 유도 (Loop)
- 영상 끝과 시작이 자연스럽게 이어지게 편집 (루프 감지 안 됨)
- 마지막 장면에 숨겨진 디테일 배치 → "다시 봐야 보이는"

## 해시태그 전략

### 조합 공식: 일반 + 중간 + 틈새 + 트렌드 (각 2-3개)

```
일반 (검색량 多): #마케팅 #B2B #그로스
중간 (경쟁 중간): #B2B마케팅 #SaaS그로스 #마케팅자동화
틈새 (우리 제품 가까움): #스타트업그로스 #세일즈퍼널
트렌드 (시즌·이벤트): #2026마케팅트렌드
```

총 8-12개가 적정. 20개 초과는 스팸 신호.

## 컴플라이언스

- 유료 광고면 자막·음성·설명란에 「광고」 또는 「유료광고」 고지
- 수치 주장 시 실증 자료 준비 (화면 내 각주·설명란 링크)
- 트렌드 오디오는 플랫폼 공식 라이브러리만 사용 (저작권)
- 미성년자 등장 시 보호자 동의·초상권 확인
