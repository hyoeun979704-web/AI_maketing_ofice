# YouTube 긴 영상 대본 플레이북

5-15분 YouTube 긴 영상 대본 구조. 도입부 15초 이탈 방지, 챕터 설계, 시청 유지 곡선(retention curve) 개선 패턴.

## 길이별 최적 구조

| 길이 | 용도 | 챕터 수 | 권장 구조 |
|------|------|---------|-----------|
| 5-7분 | 교육·튜토리얼·리뷰 | 3-4 | Cold open → Intro → 챕터 3개 → Outro |
| 8-12분 | 심층 설명·이야기·비교 | 4-6 | Cold open → Intro → 챕터 4-5개 + Mid-roll CTA → Outro |
| 13-15분 | 종합·시리즈·다큐 | 5-8 | Cold open → 2층 Intro → 챕터 6-7개 + 2개 Mid-roll → Outro |

15분 초과 영상은 완주율 급락. 분할 업로드 또는 타임스탬프 챕터화 고려.

## Cold Open (0-15초) — 최우선 제작

YouTube 이탈의 25-40%가 첫 15초에 발생. Cold open 없이 인사로 시작하면 알고리즘 신호(CTR·시청 유지)가 모두 나빠진다.

### Cold Open 5가지 타입

1. **결론 스포일러**: 영상 끝 베스트 순간 3초 프리뷰
   ```
   "이 방법으로 월 매출이 이렇게 올랐습니다 [대시보드 수치]
   방법, 지금부터 알려드릴게요."
   ```

2. **최고 장면 컷업**: 영상 내 3-5개 하이라이트 빠른 컷
   ```
   [컷 1] 시연 장면
   [컷 2] 수치 충격
   [컷 3] 고객 반응
   [컷 4] 결과
   → "자, 어떻게 했는지 전부 공개합니다."
   ```

3. **호기심 격차**: 질문 던지고 답은 나중에
   ```
   "왜 광고비 2배 써도 매출은 그대로일까요?
   이 영상 끝나면 아시게 됩니다."
   ```

4. **대답 미끼**: 구독자 질문 인용 후 답변 예고
   ```
   "'B2B 콜드 이메일, 답장율 1% 어떻게 넘기나요?'
   댓글에서 가장 많이 받은 질문입니다.
   오늘 10분 안에 다 풀어드릴게요."
   ```

5. **미니 사례**: 30초 안에 끝나는 축약 이야기
   ```
   "친구 창업자 한 명이 3개월 만에 폐업 직전에서
   매달 1천만원 매출을 만들었습니다.
   그 방법을 오늘 나눕니다."
   ```

### Cold Open 체크
- [ ] 15초 이내?
- [ ] 호기심·긴장·감정 중 하나 이상 유발?
- [ ] 구독자가 아닌 신규 시청자도 맥락 없이 이해 가능?
- [ ] 본편 내용과 실제로 일치? (낚시는 신뢰·알고리즘 모두 하락)

## Intro (15-45초)

Cold Open 직후, 호흡을 한 번 고른 뒤 본편 안내.

```
[15-25초] 인사 + 채널 소개 (5-10초, 간결하게)
"안녕하세요, OO 채널의 [이름]입니다."

[25-35초] 오늘 다룰 내용 3줄 요약
"오늘은 B2B 세일즈 이메일 3가지 유형을 소개하고,
각각 실제 답장율 데이터를 공유하고,
마지막에 누구나 쓸 수 있는 템플릿 다운로드 링크도 드립니다."

[35-45초] 구독·알림 간단 멘트 (반드시 짧게)
"영상이 도움되셨으면 구독·좋아요 부탁드려요."
```

**주의**: Intro가 45초 넘어가면 이탈률 급증. 긴 채널 소개·BGM·로고 애니메이션을 피한다.

## Body (챕터 구조)

### 챕터 설계 원칙

- 각 챕터 1-3분
- 챕터 시작마다 **장면·톤 전환** (배경·의상·BGM·화면 구도 중 하나 이상 바꿈)
- 챕터 사이에 0.5-1초 브릿지 (타이틀 카드·음향 효과)

### 챕터 시작 템플릿

```
[챕터 타이틀 카드: "1. 콜드 이메일의 3가지 유형"]
(1초 카드 + 음향)

"첫 번째 유형은 '문제 발견형'입니다."
[화면: 이메일 예시 스크린샷]

본문...
```

### 리텐션 장치 (10-30초마다)

시청 유지 그래프 상승·유지를 위한 장치.

| 장치 | 예시 |
|------|------|
| Open loop (호기심 미결) | "이 부분은 잠시 뒤에 다시 설명드릴게요" |
| 수치·데이터 플래시 | "이 방법으로 답장율이 3%에서 12%로 올랐습니다" |
| 비주얼 전환 | 새 화면·그래픽·줌인 |
| 대화형 질문 | "여러분은 어떤 방법을 써보셨나요? 댓글로 알려주세요" |
| 미니 스토리 | "실제로 이 방법으로 성공한 팀 이야기 잠깐 나눠볼게요" |

### Mid-roll CTA (4-6분 지점)

5분 이상 영상은 중반 CTA가 이탈 방지에 효과적.

```
[4-6분 사이, 챕터 간 전환 지점]
"잠깐, 다음 부분 들어가기 전에…
이런 콘텐츠가 도움되셨다면 구독·알림 꼭 눌러주세요.
다음 영상도 놓치지 않으실 수 있어요."
[즉시 본편 복귀]
```

**주의**: Mid-roll이 10초를 넘으면 역효과. 5-7초 이내.

## Outro (14-15분, 또는 마지막 30-60초)

```
[마지막 30-60초]

"오늘 영상 핵심을 한 번 더 정리해드리면…
[3개 포인트 1줄씩 빠르게]

관련해서 더 깊이 다룬 영상이 있는데, 화면에 뜨는 영상 클릭해보세요.
[엔드 카드: 다음 영상 + 구독 버튼]

도움되셨다면 구독·좋아요 한 번 눌러주시고,
다음 영상에서 또 만나요!"
```

**엔드 카드 요소**:
- 다음 영상 1개 (시청 시간 자연 연장)
- 재생목록 링크
- 구독 버튼

## 시청 유지 곡선 개선 패턴

### 이탈 지점 진단 (YouTube Studio)

1. Studio → 애널리틱스 → 참여도 → 시청 지속 시간
2. 평균 대비 큰 이탈(drop) 구간 표시 확인
3. 이탈 구간에서 무슨 일이 있었는지 타임라인 확인

### 개선 패턴별 대응

| 이탈 유형 | 원인 | 개선 |
|-----------|------|------|
| 0-15초 급락 | Cold open 약함 | 더 강한 훅·스포일러 |
| 45초 지점 급락 | Intro 길다 | 30초 이내로 압축 |
| 3-5분 완만 하락 | 한 주제 너무 길다 | 챕터 더 세분화, 비주얼 전환 |
| 4-6분 급락 | 구독 CTA가 본편 끊음 | Mid-roll 더 짧게 |
| 챕터 간 급락 | 전환이 어색 | 타이틀 카드·브릿지 넣기 |
| 마지막 1분 급락 | Outro 길다 | 요약 간결, 엔드카드만 |

### 리텐션 참고 기준

아래 수치는 업계 관찰·공개 크리에이터 리포트 기반의 대략적 참고치다. YouTube는 공식 니치별 벤치마크를 공개하지 않으므로 **채널·주제·시청자 층에 따라 크게 달라진다.** 본인 채널의 YouTube Studio 애널리틱스에서 비교 그룹 대비 평균을 먼저 확인한 뒤, 거기서 +5-10% 개선을 목표로 삼는 것이 현실적.

일반적 기준점 (참고):
- 교육·정보형 8분 영상 평균 유지율: 40-50% 내외
- 첫 30초 유지율: 55-65% 수준이 양호
- 중간 시점(50%) 유지율: 45-55% 수준이 양호

**실행**: 본인 영상 데이터 수집 → 3-5개 비교 그룹 벤치마크 → 취약 구간부터 개선. 절대 수치 추종보다 상대 개선이 중요.

## SEO·제목·썸네일 (영상 성과 50% 좌우)

### 제목 공식
- 주 키워드 앞쪽 배치
- 숫자·연도·한정 표현 포함 ("2026", "3가지", "단 7분에")
- 60자 이내 (모바일 컷오프)
- 과장·낚시 금지 (노출은 되지만 리텐션 파괴)

### 썸네일 공식
- 얼굴 클로즈업 (감정 과장) + 6자 이내 키워드
- 색상 대비 강하게 (오렌지·노랑·빨강 눈에 잘 띔)
- 화살표·원·박스 등 유도 시각
- A/B 3개 제작, Studio 테스트

### 설명란 구조

```
[첫 100자: 영상 핵심 요약·키워드 자연 배치]

[타임스탬프]
00:00 인트로
00:45 챕터 1
03:12 챕터 2
...

[관련 링크]
- 본 영상에서 언급한 템플릿: [링크]
- 관련 영상: [링크]
- 채널 소개 / SNS

[해시태그 5-10개]
#마케팅 #B2B #콜드이메일
```
