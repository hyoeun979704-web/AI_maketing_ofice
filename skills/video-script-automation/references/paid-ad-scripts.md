# 유료 광고 영상 대본 템플릿

Meta·YouTube·TikTok 광고용 15-60초 대본 구조, 플랫폼별 규격, 후킹 문장 스왑 파일.

## 플랫폼별 규격 요약

| 항목 | Meta Reels/Feed | YouTube In-stream | YouTube Shorts Ads | TikTok Ads |
|------|-----------------|-------------------|---------------------|------------|
| 화면 비율 | 9:16 / 1:1 / 4:5 | 16:9 (세로도 지원) | 9:16 | 9:16 |
| 권장 길이 | 9-15초 (Reels), 15-30초 (Feed) | 6초 범퍼 / 15-30초 skippable | 6-60초 | 9-15초 |
| 첫 노출 중요 | 3초 이내 Hook | 5초 (skip 직전) | 3초 | 3초 |
| 자막 | 무음 재생 기본, 필수 | 필요 (CC) | 권장 | 필수 |
| 사운드 | 무음 전제 | 음량 낮음 전제 | 옵션 | 트렌드 오디오 활용 |
| CTA | 「지금 보기」·「구매」 버튼 | 카드·엔드스크린 | 「자세히」 | 「지금 보기」 |

## 핵심 구조: Hook-Problem-Solution-CTA (HPSC)

모든 짧은 광고 영상의 기본 골격. 15초 영상 기준 초당 1초 = 1문장 원칙.

```
0-3초:   Hook (주목)
3-6초:   Problem (공감)
6-12초:  Solution (해결·제품)
12-15초: CTA (행동)
```

30초 이상 광고는 Solution을 6-18초로 확장하고 증거(데이터·후기·시연) 추가.

## Hook 라이브러리 (3초 이내 훅)

### 질문형
- "매출이 늘긴 느는데, 왜 이익은 그대로일까요?"
- "월 100만원으로 B2B 리드 100개 가능할까요?"
- "당신의 세일즈 팀, 매주 몇 시간을 입력에 쓰고 있나요?"

### 수치 충격
- "3개월 만에 MRR 2배, 우리가 한 딱 3가지."
- "업무의 40%가 사라졌습니다. 이 도구 하나로."
- "광고비를 2분의 1로 줄이고 전환을 1.5배 올린 방법."

### 역발상
- "세일즈는 말을 덜 할수록 잘 됩니다."
- "이메일 오픈율, 제목이 아니라 발송 시간 문제입니다."
- "고객은 기능이 아니라 '변화한 자신'을 삽니다."

### 호기심 격차
- "이거 모르면 매달 300만원 손해보고 있습니다."
- "경쟁사 절반이 이미 쓰고 있는 이 툴, 아직 모르시나요?"
- "상위 1% SaaS만 쓰는 마케팅 프레임워크."

### 즉시 유용
- "지금 당장 엑셀 없이 할 수 있는 3가지."
- "무료로 시작하는 가장 빠른 그로스 해킹."
- "30초 안에 설정 끝, 다음 달부터 매출 보입니다."

### 감정 공감
- "이번 분기, 목표 못 채웠죠? 저도 그랬어요."
- "야근하고 퇴근하는 길이 무거워지는 그 느낌."

## Problem 섹션 (3-6초)

타겟의 구체적 고통점을 **시각·청각으로 압축**.

- 화면: 카드 청구서·막힌 엑셀·늦게까지 불 켜진 사무실
- 자막: "새는 돈, 보이지 않죠"
- 내레이션: "매달 청구서 받고 한숨 쉬는 그 느낌."

Problem은 1문장·3초 안에 끝낸다. 길어지면 Solution 공간 부족.

## Solution 섹션 (6-12초)

제품·서비스가 Problem을 어떻게 해결하는지 **1-2개 핵심만**.

- 기능 나열 X, 사용 장면 + 결과 O
- "자동으로 OO해줍니다" → 자동화되는 화면 시연
- 수치 증거는 실증 자료 필요 (컴플라이언스)

```
화면: 대시보드에 숫자 올라가는 애니메이션
자막: "한 번 설정 → 자동 정리 → 실시간 리포트"
내레이션: "매일 30분이 3분으로. 월 15시간 절약."
```

## CTA 섹션 (12-15초)

구체적·즉시 실행 가능한 다음 행동.

### 약한 CTA (신중히)
- "더 알아보기"
- "링크 참고"

### 강한 CTA (권장)
- "지금 무료로 시작 →"
- "14일 체험, 카드 정보 없이 →"
- "댓글로 '예산' 남기시면 맞춤 견적 보내드려요."
- "오늘까지 가입하면 첫 달 반값."

### 한국 광고 표기 (필수)
- 영상 좌상단 또는 하단에 「광고」 라벨 상시 노출
- 설명란에 `#광고` `#유료광고` 해시태그
- 협찬·뒷광고 방지 가이드라인 준수 (공정위 표시·광고 심사지침)

## 완성 예시

### 예시 A: B2B SaaS (Meta Reels 15초)

```
[0-3초] Hook
화면: 노트북 앞에 무표정한 직장인, 시계는 밤 10시
자막: "왜 일은 끝나지 않는가"
내레이션: "야근해도 숫자가 안 보이는 그 느낌."

[3-6초] Problem
화면: 엑셀 수십 개 탭 빠르게 전환
자막: "데이터는 쌓이는데 인사이트는 없다"
내레이션: "스프레드시트만 한 달을 채워도."

[6-12초] Solution
화면: 제품 대시보드 클로즈업, 자동 리포트 생성 애니메이션
자막: "자동으로 정리되고 → 바로 보이는 인사이트"
내레이션: "한 번 연결, 매주 자동 리포트. 실제 2,000개 팀이 선택한 방식."

[12-15초] CTA
화면: "무료로 시작" 버튼 클로즈업
자막: "14일 무료, 카드 없이"
내레이션: "지금 시작하세요."

자막 고정: 「광고」 (상단 좌측)
설명란: "#광고 #B2BSaaS #마케팅자동화"
```

### 예시 B: 이커머스 (TikTok 9초)

```
[0-3초] Hook + Problem 동시
화면: 집 옷장 문 열자 옷 무너져 내림
자막: "옷장이 이런 당신"
내레이션: "아침마다 이 지옥."

[3-6초] Solution
화면: 제품 (슬림 옷걸이) 적용 전후 비교
자막: "이 옷걸이 하나로"
내레이션: "공간은 2배, 꺼내기 3배 빠르게."

[6-9초] CTA
화면: 제품 + 가격 + "오늘만 50% 할인"
자막: "지금 쇼핑 →"
내레이션: "링크 프로필에."

자막 고정: 「광고」
```

### 예시 C: YouTube 6초 범퍼 광고

```
[0-6초 통합]
화면: 문제 상황 1초 + 제품 3초 + 로고·CTA 2초
자막: "커피 한 잔 값으로 매일 자동화"
내레이션: "OO 지금 시작. 첫 달 무료."

고정: 「광고」
```

## 바리에이션 스왑 파일

같은 상품·문제·해결책으로 **Hook만 바꾼 3-5개 버전**을 만들어 A/B 테스트 ([ab-test-setup] 스킬 연계).

```
버전 1 (질문 훅): "왜 일은 끝나지 않는가?"
버전 2 (수치 훅): "주 15시간 절약한 팀의 비결."
버전 3 (역발상): "리포트는 직접 만들지 마세요."
버전 4 (호기심): "상위 1% 팀만 쓰는 이 방법."
버전 5 (공감): "퇴근이 늦어지는 그 느낌."
```

Problem·Solution·CTA는 공유, Hook만 교체. 퍼포먼스 광고에서 CTR 차이가 가장 크게 나는 부분이 Hook.

## 체크리스트

### 제작 전
- [ ] 타겟 오디언스 명시 (product-marketing-context 참조)
- [ ] 플랫폼·규격 결정
- [ ] Hook 3-5개 대안 작성
- [ ] 실증 자료 준비 (수치 주장이 있는 경우)

### 제작 중
- [ ] 무음 재생 가정 → 자막 필수
- [ ] 3초 룰: 핵심 메시지 전달 완료
- [ ] 한 장면 ≤3초 (컷 빈도)
- [ ] 한국 광고 표기 「광고」 고정

### 제작 후
- [ ] 설명란 #광고 해시태그
- [ ] 실증 자료 링크·URL 첨부
- [ ] 사운드 on/off 모두 테스트
- [ ] 여러 사이즈로 리사이즈 (9:16, 1:1, 4:5, 16:9)
- [ ] 플랫폼별 업로드 규격 확인
