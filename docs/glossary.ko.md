# Marketing Glossary (영-한 대조표)

모든 한국어 번역은 이 글로서리의 용어를 따른다. 수정은 PR로 제안.

## 사용 원칙

1. **선택 기준**: 실무에서 널리 쓰이는 표현 우선. 원어 그대로 쓰는 용어는 그대로 음차 (예: "랜딩페이지", "카피라이팅")
2. **첫 등장 시 원어 병기**: 문서에서 처음 나오는 용어는 "랜딩페이지(landing page)" 형태로 표기
3. **고정 명사는 유지**: 제품명·플랫폼명(Naver, Kakao, Stripe, GA4)은 번역하지 않음
4. **약어 취급**: 영문 약어는 그대로 사용 (CRO, CTR, CAC, LTV, ICP, SEO)
